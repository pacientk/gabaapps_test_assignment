## Component Specifications

### UserCard
```tsx
// States: default | hover | focused | loading(skeleton)
// Size: full width of grid column
// Aspect: auto height

interface UserCardProps {
  user: User
  onClick: (userId: number) => void
  isSelected?: boolean  // highlighted when drawer is open
}
```

Visual hierarchy:
1. Top row: Avatar (left) + Name + Role badge (right aligned)
2. Job title (secondary text)
3. Department · Company (muted, with separator dot)
4. Email (with mail icon)
5. City, Country (with map-pin icon)

Hover state: `border-blue-300 shadow-md` + subtle scale `scale-[1.01]`
Selected state: `border-blue-500 ring-2 ring-blue-100`

---

### UserAvatar
```tsx
interface UserAvatarProps {
  src: string
  name: string          // for alt text + fallback initials
  size?: 'sm' | 'md' | 'lg'  // 32 | 48 | 80px
}
```
- Shows image with Next.js `<Image>` (optimized)
- Falls back to initials in colored circle on image error
- Initials color derived from name hash (deterministic)

```tsx

  value: string
```

## Accessibility Requirements
- All interactive elements: visible focus ring (`focus-visible:ring-2 ring-blue-500`)
- Images: descriptive `alt` text
- Drawer: `role="dialog"` + `aria-label` + focus trap
- Filter dropdowns: proper `aria-expanded`, `aria-haspopup`
- Search: `role="search"` wrapper, `aria-label` on input
- Loading states: `aria-busy="true"` on container
- Error states: `role="alert"` on error message
- Pagination: `aria-label="Pagination"` + `aria-current="page"` on active page

## Animation Spec
```
Card hover:     transition-all duration-150 ease-out
Drawer open:    slide from right, duration-300 ease-out (translateX: 100% → 0)
Drawer close:   slide to right, duration-200 ease-in
Skeleton pulse: animate-pulse (Tailwind built-in)
Filter chip:    fade in, duration-150
```
Use `@radix-ui/react-dialog` for drawer — handles animations via data attributes.
